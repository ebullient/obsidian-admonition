# Create an Admonition

To create an Admonition, navigate to the [Admonition Settings](settings.md#add-new).

</details>

## Steps

1. Navigate to Obsidian's Settings.
2. Select Admonitions from the Plugin List.
3. Select **Add New** from the first set of options in [Settings](settings.md).
4. In the new window that opens:
   1. Enter an Admonition Type. This will be the name used to call it in code block or callout form.
   2. If desired, choose a default title.
   3. If desired, opt out of no title by default. Enabling this will require a `title:` at the top of the admonition to create a title.
   4. If desired, enable the copy button.
   5. Select an admonition icon. You can use your own or select one from the included icon packs (see [Icon](options.md#icon)).
   6. Choose whether the color is created via CSS or stored within `.json`. If choosing the `.json` option, also select your color from the color picker.
